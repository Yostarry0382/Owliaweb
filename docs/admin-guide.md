## ログイン方法

1. `admin.html` にアクセスします
2. パスワード入力欄にパスワードを入力します
3. 「ログイン」ボタンをクリック（またはEnterキー）でダッシュボードに入れます

> ログアウトするには、ダッシュボード右上の「ログアウト」ボタンをクリックします。

---

## ダッシュボード概要

ログイン後、3つのタブで管理機能にアクセスできます。

| タブ | 機能 |
|------|------|
| 💬 ユーザー要望 | トップページから送信された要望の一覧管理 |
| 📊 クリック統計 | アプリごとのクリック数と履歴の確認 |
| 📝 コンテンツ管理 | アプリ情報・サイト文言の編集 |

---

## ユーザー要望タブ

トップページのフォームからユーザーが送信した要望がカード形式で一覧表示されます。

- **件数バッジ**: ツールバーに現在の要望件数が表示されます
- **削除**: 各カード右上の「✕」ボタンで個別に削除できます
- **データ保存先**: ブラウザの localStorage（`owlia_requests`）に保存されています

> 要望が0件の場合は「まだ要望はありません」と表示されます。

---

## クリック統計タブ

サーバーAPI（`api/clicks.php`）から取得したクリックデータを表示します。

### 表示内容

- **サマリーカード**: アプリごとの総クリック数をカード形式で表示
- **クリック履歴テーブル**: 最新50件のクリック履歴を「アプリ名」「日時」の表で表示

> サーバーに接続できない場合は「サーバーからデータを取得できませんでした」と表示されます。

## コンテンツ管理タブ

トップページに表示されるアプリ情報とサイト文言を管理画面から直接編集できます。
「アプリ管理」と「サイト文言」の2つのサブタブで構成されています。

---

### アプリ管理

アプリの一覧を編集・追加・削除できます。

#### ツールバーのボタン

| ボタン | 機能 |
|--------|------|
| **+ アプリ追加** | 新しいアプリをリストの末尾に追加 |
| **💾 保存** | 現在の編集内容を localStorage に保存 |
| **👁 プレビュー** | 保存してトップページを新しいタブで開く |
| **↩ デフォルトに戻す** | localStorage のオーバーライドを削除し、`data/apps.js` の初期データに戻す |
| **⬇ JSONエクスポート** | 現在のアプリデータを `apps.json` としてダウンロード |

#### アプリカードの操作

- カードのヘッダー部分をクリックすると編集フォームを展開/折りたたみできます
- 右上の「✕」ボタンでアプリを削除できます（確認ダイアログが表示されます）

#### 編集可能なフィールド（基本情報）

| フィールド | 説明 | 例 |
|------------|------|-----|
| ID | アプリの一意識別子 | `sprite` |
| アプリ名 | 表示名 | `Owlia-Sprite` |
| 役割 | 短い役割説明 | `常駐型AIエージェント` |
| 説明 | カードに表示される説明文 | `デスクトップ常駐AIエージェント。...` |
| バッジ | 利用状態の選択 | `✅ 利用可能` / `⏳ Coming Soon` |
| アイコン画像パス | アイコン画像のファイルパス | `docs/character/transparent/sprite.webp` |
| カテゴリ | フィルタ用カテゴリ（複数選択可） | `利用可能`, `開発向け`, `全職種向け`, `Coming Soon` |
| タグ | カンマ区切りのタグ | `ショートカットキー起動, 音声入力` |
| CTAラベル | ボタンのテキスト | `ダウンロード` |
| CTAリンク | ボタンのリンク先URL | `go.html?app=sprite&url=...` |

#### 編集可能なフィールド（詳細ページ）

| フィールド | 説明 |
|------------|------|
| 絵文字 | 詳細ページで表示する絵文字 |
| アイキャッチ画像 | 詳細ページのメイン画像パス |
| 役割（詳細） | 詳細ページの役割テキスト |
| 説明（詳細） | 詳細ページの長い説明文 |
| 機能一覧 | 1行1項目で機能を列記 |
| ヒント（HTML可） | 補足情報（HTMLタグ使用可） |
| 詳細CTAラベル | 詳細ページのCTAボタンテキスト |

### サイト文言

トップページの各セクションに表示されるテキストを編集できます。

#### ツールバーのボタン

| ボタン | 機能 |
|--------|------|
| **💾 保存** | 編集内容を localStorage に保存 |
| **👁 プレビュー** | 保存してトップページを新しいタブで開く |
| **↩ デフォルトに戻す** | `data/site.js` の初期データに戻す |
| **⬇ JSONエクスポート** | 現在のサイト文言を `site.json` としてダウンロード |

#### 編集可能なセクション

| セクション | フィールド | 初期値の例 |
|------------|------------|------------|
| **ヒーロー** | サブタイトル | `NIT社内生成AIプラットフォーム` |
| | メインタイトル | `Owlia` |
| | キャッチコピー | `あなたに寄り添う愉快な仲間たち` |
| **APPSセクション** | ラベル / タイトル / サブタイトル | `📦 APPS` / `やりたいことから選ぶ` / ... |
| **FOR YOUセクション** | ラベル / タイトル / サブタイトル | `👤 FOR YOU` / `あなたに合うアプリ` / ... |
| **DETAILSセクション** | ラベル / タイトル / サブタイトル | `🔍 DETAILS` / `各アプリの詳しい使い方` / ... |
| **CTAセクション** | ラベル / タイトル / サブタイトル | `GET STARTED` / `今日からOwliaを始めよう` / ... |
| **フッター** | テキスト | `NIT社内生成AIプラットフォーム — produced by dAIrab` |

## データの仕組み

### データフロー

```
data/apps.js, data/site.js  (デフォルトデータ)
        ↓
  OwliaContent.init()  (content-loader.js)
        ↓
  localStorage にオーバーライドがあれば上書き
        ↓
  トップページ / 管理画面で使用
```

### 保存先

| データ | localStorage キー | デフォルトデータ |
|--------|-------------------|------------------|
| アプリ情報 | `owlia_cms_apps` | `data/apps.js` |
| サイト文言 | `owlia_cms_site` | `data/site.js` |
| ユーザー要望 | `owlia_requests` | なし |

### 注意事項

- データは**ブラウザの localStorage** に保存されるため、同じブラウザでのみ反映されます
- 「デフォルトに戻す」を実行すると、localStorage のオーバーライドが削除され、`data/` フォルダ内のJSファイルのデータが使われます
- 恒久的にデータを変更したい場合は、JSONエクスポートしたデータを `data/apps.js` や `data/site.js` に反映してください

## よくある質問

### Q: 編集した内容が他の人に反映されない

A: コンテンツ管理の変更は localStorage に保存されるため、編集したブラウザでのみ有効です。全員に反映するには、JSONエクスポートしたデータを `data/apps.js` / `data/site.js` に反映してデプロイしてください。

### Q: 間違えてデータを消してしまった

A: 「デフォルトに戻す」ボタンで `data/` フォルダ内の初期データに復元できます。ただし、localStorage 上のカスタマイズはすべて消去されます。

### Q: オーバーライドが適用されているか確認したい

A: コンテンツ管理タブを開いた際、黄色の通知バー「⚠️ ローカルストレージのオーバーライドが適用されています」が表示されていれば、localStorage のデータが使われています。

### Q: クリック統計が表示されない

A: クリック統計はサーバーAPI（`api/clicks.php`）からデータを取得します。サーバーが起動していない、またはAPIエンドポイントにアクセスできない場合はデータが表示されません。
